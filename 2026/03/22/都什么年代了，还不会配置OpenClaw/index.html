前言

最近 OpenClaw 在各大平台爆火，也就是大家常说的“养龙虾”。然而我发现很多小伙伴连最基本的配置都不会，更令人惊讶的是洛谷上竟然还没有相关的教程。作为一名先行者，我决定写下这篇教程，希望能帮助到大家。

准备工作

在开始之前，需要先安装以下两个工具：

  • Git（版本控制工具）
  • NodeJS（JavaScript 运行环境）

具体下载和安装方法请自行搜索相关资料，这里不再赘述。

1. 以管理员身份打开 PowerShell

二、配置 OpenClaw

安装完成后，程序会自动进入配置向导，按照以下步骤操作即可：

1. 初始设置

  • 当出现风险提示对话框时，选择 Yes（接受风险）
  • 接着选择 QuickStart（快速启动模式），除非你是专业用户需要自定义配置

2. 核心配置

  • 选择 AI 模型提供商：根据自己的偏好选择（如 OpenAI、Azure 等）
  • 输入 API 密钥：将对应平台的 API 密钥粘贴进去
  • 选择具体模型：根据需求选择合适的版本
  • 配置通讯通道：
    • 如果暂时不需要，可以选择 skip for now（稍后配置）
    • 个人推荐使用飞书（后续可能会出一篇专门讲解飞书配置的文章）

3. Skills 配置

系统会询问是否安装 skills（功能插件）：

  • 如果你有明确的使用目的，可以选择对应的 skill
  • 如果不确定，也可以之后再进行配置

4. 其他选项

  • API 相关选项：除非你确实有对应服务商的 API，否则全部选择 No
  • Hook 功能（定时检查插件）：
    • ⚠️ 风险提示：据说国外有位小哥因为这个功能直接损失好几百美元
    • 建议：除非有特殊需求，否则不选择此功能

5. 完成配置

配置完成后，系统会自动：

  1. 弹出一个新的命令行窗口（无需操作，让它自动运行）
  2. 在默认浏览器中打开一个网页界面

验证是否成功：

  • 在网页界面向 OpenClaw 发送消息
  • 如果能收到回复，恭喜你！配置成功啦 awa
